🔒 隐私政策

简单直白，不跟你绕弯子

📌 一句话总结

本站不主动收集你的任何个人信息。你没有账号、不需要登录、也不需要填任何东西才能看攻略。但是——托管平台和第三方工具会自动收集一些基础数据，下面详细说明。

🖥️ 托管服务：GitHub Pages

本站托管在 GitHub Pages，所有页面文件都放在 GitHub 的服务器上。当你访问本站时，GitHub 会自动记录：

  • 你的 IP 地址
  • 访问时间和页面路径
  • 浏览器类型（User-Agent）

这些数据归 GitHub 管，我们看不到也没法改。详见 GitHub 隐私声明。

📊 访客统计：Microsoft Clarity

本站用 Microsoft Clarity 做行为分析，了解访客看了哪些页面、点了什么。Clarity 收集的是：

  • 页面浏览记录（你看了哪些页面、看了多久）
  • 点击和滚动行为（你点了哪个链接、滚了多远）
  • 设备信息（屏幕尺寸、浏览器类型、操作系统）

Clarity 不收集可识别个人身份的信息（不记录 IP、不记录键盘输入、不记录表单内容）。详见 Microsoft 隐私声明。

📋 反馈表单：腾讯问卷

本站通过 腾讯问卷（wj.qq.com）收集反馈。你提交反馈时，腾讯问卷会按自己的隐私政策处理数据，我们这边只能看到你填的内容。

反馈表单是可选的——你不填也不影响看攻略。

🍪 本地存储（LocalStorage）

本站用浏览器的 localStorage 记住你的主题偏好（暗色/亮色模式）。这个数据：

  • 只存在你自己的浏览器里
  • 不会被上传到任何服务器
  • 清除浏览器缓存就没了

本站不使用任何自建 Cookie。但第三方服务（见下方）可能使用它们自己的 Cookie。

📢 广告：Google AdSense

本站使用 Google AdSense 展示广告。Google 会使用 Cookie 来投放与你兴趣相关的广告。这些 Cookie 由 Google 控制，不是我们控制的。

  • Google 使用 DoubleClick DART Cookie 来根据你的浏览历史展示个性化广告
  • 你可以在 Google 广告设置 中关闭个性化广告
  • 也可以在 Network Advertising Initiative 统一退订多家广告网络的 Cookie

第三方供应商（包括 Google）使用的 Cookie 详情见 Google 广告技术说明。

🔗 外部链接

本站包含指向外部网站的链接（如 Steam、Ludeon Studios、Mod 页面等）。点这些链接后，你在那些网站的隐私受它们各自的隐私政策保护，跟本站没关系。

⚖️ 免责声明

  • 本站攻略内容基于 RimWorld 1.4+ 全DLC 版本，后续版本更新可能导致部分内容过时
  • RimWorld 游戏及其所有 DLC 的版权归 Ludeon Studios 所有。本站是玩家社区自制攻略，与 Ludeon Studios 无关
  • 本站提到的 Mod 均为其各自作者的版权，推荐仅代表个人使用经验
