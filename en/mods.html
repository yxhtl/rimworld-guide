ModCategoryDescriptionRecommendation
HarmonyFramework99% of mods depend on it. Low-level library for hooking game methods.Essential, place first
HugsLibFrameworkGeneral-purpose library providing API support for other mods.Essential, place second
Vanilla Expanded FrameworkFrameworkCore dependency for the entire Vanilla Expanded series.Required if using VE series

💡 Veteran's Advice: More mods isn't always better. Always back up your save before adding mods. With over 100 mods, startup can take 5–10 minutes — don't panic thinking it's frozen.

5. Quick Load Order Reference —— Ordered top to bottom

  1. Harmony — Very top, place first
  2. Core — Game core
  3. DLC — Royalty → Ideology → Biotech → Anomaly
  4. HugsLib
  5. Vanilla Expanded Framework
  6. Large Framework Mods — such as JecsTools
  7. Gameplay Mods — UI / Behavior
  8. Content Additions — New items / Creatures / Races
  9. Localization / Translation Patches
  10. Compatibility Patches — Bottom
The mantra: Frameworks on top, content below; the more fundamental, the higher; the more superficial, the lower
