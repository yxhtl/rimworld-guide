🧩 Mod 推荐

1. 必装前置 Mod —— 不打这些，大部分 Mod 跑不起来

Mod分类说明建议
Harmony前置框架99% 的 Mod 都依赖它，Hook 游戏方法的底层库必装，放第一位
HugsLib前置框架通用功能库，给其他 Mod 提供 API 支持必装，第二位
Vanilla Expanded Framework前置框架Vanilla Expanded 系列全家桶的底层依赖用 VE 系列就必装

2. UI 优化类 —— 不改玩法，纯体验提升

Mod功能推荐指数
RimHUD不用点开小人就能看到详细属性面板必装 省 90% 点击
Allow Tool一键允许所有、选择相似物品、紧急搬运标记必装 操作效率翻倍
Numbers列表视图查看所有小人的技能/心情/健康数据推荐 管理神器
Replace Stuff升级墙体不用先拆后盖，直接覆盖材料替换推荐 省人工
Quality Builder指定建造品质，造出来低于要求自动拆了重盖推荐 强迫症必备
Blueprints蓝图复制粘贴，对称建造不求人推荐 规划神器
Color Coded Mood Bar心情条按颜色区分，一眼看出谁要崩了推荐 心情管理

3. 功能扩展类

Mod功能推荐指数
Wall Light墙上装灯，不占地板空间必装 每个房间必备
Pick Up And Haul小人一次搬多个物品，用背包逻辑替代单件搬运必装 搬运效率质变
Common Sense小人更聪明了：做菜前先扫地、睡觉前先娱乐推荐 AI 优化
RunAndGun边跑边打，不用站桩射击推荐 提升战术深度
Simple Sidearms小人能带多把武器，自动切换近战/远程推荐 战斗质变
Dubs Bad Hygiene加入卫生/水管/厕所系统，殖民者要上厕所洗澡可选 增加真实感
Rimefeller石油工业体系：钻油→炼油→塑料→发电可选 后期内容扩展

4. 内容扩展类

Mod功能推荐指数
Vanilla Expanded 系列大量新武器、护甲、家具、植物、动物，风格贴合原版必装 量大管饱
Combat Extended完全重做战斗系统——弹药、弹道、穿透、护甲机制推荐 战斗党必装，注意兼容性
Alpha Animals100+ 新生物，从可爱到恐怖，生态多样化推荐 生态多样化
Android Tiers机器人种族，可制造可控制，科幻感拉满可选 科幻感拉满
Save Our Ship 2飞船变移动基地，太空战斗 + 探索，通关后的新游戏推荐 通关后新游戏
💡 老手忠告： Mod 不是越多越好，每次加 Mod 前记得备份存档。Mod 数量超过 100 个时启动可能需要 5–10 分钟——别以为卡死了。

5. 加载顺序速查 —— 从上到下排列

  1. Harmony — 最顶，放第一位
  2. Core — 游戏本体
  3. DLC — Royalty → Ideology → Biotech → Anomaly
  4. HugsLib
  5. Vanilla Expanded Framework
  6. 大型前置框架 — 如 JecsTools
  7. 功能修改 — UI / 行为类
  8. 内容添加 — 新物品 / 生物 / 种族
  9. 汉化包
  10. 兼容补丁 — 最底
口诀：前置在上，内容在下；越底层越靠上，越表面越靠下
