🚀 开局生存指南

1. 选人 —— 三个人决定你的命运

开局选人是最重要的一步，比落地地点还关键。三个小人的技能组合直接决定你能不能活过头三天。


角色必备技能绝对不能有
建造师建造 ≥ 8，手工 ≥ 5懒虫、工作怠慢
种植/烹饪种植 ≥ 8，烹饪 ≥ 6纵火狂（会烧你家！）
研究员/医生智识 ≥ 8，医疗 ≥ 6工作怠慢、崩溃艺术家
⚠️ 绝对避雷：纵火狂（Pyromaniac）、意志动荡（Volatile）、工作怠慢（Lazy）——这三个特性直接重roll，别犹豫。
✅ 加分特性：过目不忘（Fast Learner）、意志如铁（Iron-Willed）、工作狂（Industrious）——见到直接锁。

2. 落地 —— 选对地方成功一半

  • 地形：新手选温带森林 或 干旱灌木林，别碰冰原和沙漠
  • 山地 vs 平原：山地易守难攻但会刷虫灾，平原好盖房但防守面大——新手推荐大丘陵（有山但不多）
  • 生长季：至少选 30/60 天（全年生长更好），否则种不出吃的
  • 石头类型：优先 花岗岩 > 石灰岩 > 砂岩（花岗岩墙体最硬，= 你活得久）
  • 派系距离：别离海盗太近，友善部落近一点方便贸易

3. 前 3 天行动清单

顺序做什么为什么
第1步划种植区 + 播种水稻水稻 3 天就能收第一茬，没吃的你啥都干不了
第2步盖一间 7×7 木屋 + 屋顶当临时仓库和睡觉用，别盖太大——后面要换石头
第3步放 3 个睡眠点 + 篝火小人睡地上不掉心情，但睡外面掉；篝火防止冻伤
第4步屠宰落地动物 + 做简易餐打包食品只能撑几天，先杀落地就有的动物
第5步划仓库（室内）+搬物资药品、食物、零件放外面会坏，赶紧搬进屋
第6步做研究桌 + 开始研究第一优先级科技：电池 → 太阳能 → 制冷（冰箱）
💡 关键认知：RimWorld 的核心循环是"吃 → 住 → 防 → 研究"。第一要务永远是食物来源稳定，第二才是盖房子。

4. 第一周：从活下来到站稳脚

  • 造屠宰台 + 灶台：简易餐吃不久，尽快做精致餐（需要灶台 + 素食 + 肉/蛋）
  • 划 3 间独立卧室：有独立房间 +5 心情，比睡大通铺强太多
  • 做石匠桌 + 切石头：木头墙不耐火（闪电一下全家烧没），尽快换石头墙
  • 造马蹄铁桩：最便宜的娱乐设施，防止小人"无聊"掉心情
  • 划垃圾储存区：尸体、石头渣子丢远点，别堆在屋里
