🔬 科技研究路线

📋 研究路线总览（新手标准开局）

适用于 劫掠逃生 (Crashlanded) 开局，已经给你排好优先级了，照着点就行：

科技名称顺序为什么先研究它
电池 (Batteries)第1个没电池晚上就断电，太阳能板白天发的电存不住等于白搭
太阳能板 (Solar Panels)第2个稳定的自发电来源，摆脱对地热蒸汽间歇泉的位置依赖
制冷 (Air Conditioning)第3个冷库=存粮，没有冷库你的食物撑不过第一个冬天
石匠 (Stonecutting)第4个木头房一把火就没了，石头墙才是永远的家
微电子基础 (Microelectronics)第5个解锁高级研究桌和通讯台，能叫商队能摇人
精密装配 (Precision Machining)第6个能做零件了，修东西修设备不用再等路过的贸易商队
高级研究 (Advanced Research)第7个解锁仿生学和飞船科技，正式进入后期
地热发电 (Geothermal)第8个24小时稳定3600W，一个地热机够大半个基地用
💡 核心逻辑：先保证有电 → 再保证有饭吃 → 然后升级材料 → 最后冲高科技。这条线别跳步，跳了就得返工。

🔄 不同开局的研究路线差异

三种开局节奏完全不同，别拿标准开局的路线往部落局里套：

开局类型推荐研究策略核心差异
标准开局 (Crashlanded) 电力 → 冷库 → 石匠 → 微电子 → 精密装配 三人开局，自带基础科技，稳稳按标准路线走就行
部落开局 (Tribal) 复杂衣物 → 复杂家具 → 恶魔素 → 冲电力 研究速度极慢，别想着一上来搞电。先点出防寒衣物和家具保证生存，恶魔素研究出精神茶（啤酒替代品）撑心情，最后才碰电力
富探险家 (Rich Explorer) 微电子基础 → 高级研究 → 精密装配 自带部分科技和资源，直接跳基础冲后期，快速形成装备碾压

🎯 关键科技节点速查

这些科技是真正的分水岭，点到就是质变：

科技名称解锁内容为什么重要
制冷冷库建造食物保存从几天变几年。不研究冷库？冬天等着吃老鼠吧
石匠石材加工石头墙防火，再也不用担心一记闪电把你家烧光
微电子基础通讯台 + 高级研究桌通讯台能呼叫商队和盟友支援，可以说是保命技能
精密装配零件制作自己能做零件了，不用跪求贸易商队，修理和建造都自由了
地热发电机地热发电24小时稳定3600W，再也不用看风力和太阳能的脸色
水培 (Hydroponics)室内种植冰原、沙漠也能量产粮食，地盘自给自足
仿生体 (Bionics)仿生部件制作给小人换零件，移动速度和战斗力直接质变
制造技术 (Fabrication)高级零部件制作能做高级装备 = 装备自由，不再依赖掉落和商队
星际航行基础 (Starship)飞船建造造飞船 = 通关条件，这就是最终目标

⚡ 研究效率提升技巧

同样的时间，有人研究快一倍——区别全在这些细节里：

  • 研究桌放 无菌地砖 上，干净环境提升研究速度，脏乱差会拖慢进度
  • 多重分析仪 (Multi-analyzer) 连到研究桌，研究速度 +10%，造一个不亏
  • 重点 安排一个专职研究员，别让研究位兼建筑挖矿——专注才是最快的速度
  • 智识 ≥ 8 的小人研究效率明显提升，低于 5 的别让他碰研究桌，纯属浪费时间
  • 夜里研究工作，桌旁放盏灯。黑暗中研究速度 -20%，一盏灯的事别省
  • 后期可以造 多个研究桌 + 多个研究员并行研究，科技树直接起飞
  • 研究加速药剂 (Research Accelerator) 有就用，不用留着过年
  • 研究重要科技前，把全基地智识最高的人调去专职研究，最大化效率
